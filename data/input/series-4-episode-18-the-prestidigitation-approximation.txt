		<h2 class="title">Series 4 Episode 18 &#8211; The Prestidigitation Approximation</h2>
			<div class="entrytext">
				<p><em>Scene: Leonard’s bedroom</em></p>
<p>Priya: Morning.</p>
<p>Leonard: Morning.</p>
<p>Priya: Sleep okay?</p>
<p>Leonard: Mm-hmm. Great. Hey, can I ask you something?</p>
<p>Priya: Sure.</p>
<p>Leonard: Last night, me wanting to try that stuff out of the Kama Sutra, was that fun for you or kinda racially insensitive?</p>
<p>Priya: Yeah, just because you&#8217;re in bed with an Indian woman, you think that gives you permission to use crazy positions from an ancient Indian love manual?</p>
<p>Leonard: Hey, if you can find a book called weird sex with white boys, I&#8217;d be okay with that.</p>
<p>Priya: No, you have such beautiful eyes. Have you ever thought about getting contacts?</p>
<p>Leonard: I tried in the seventh grade. I could never get used to them.</p>
<p>Priya: Oh, that&#8217;s too bad.</p>
<p>Leonard: Yeah, if I had contacts I would have been the coolest debate club president ever to be stuffed into his own cello case.</p>
<p>Priya: If you had them on now, you could see what we&#8217;re going to do next.</p>
<p>Leonard: Th-th-th-that&#8217;s okay. I can infer from context.</p>
<p><em>Scene: The cafeteria.</em></p>
<p>Howard: Hey, Raj, wanna see a new magic trick I&#8217;ve been working on?</p>
<p>Sheldon: Howard, if I may interject here with a piece of friendly advice. Is working on magic tricks really how you want to spend your time? Granted, you&#8217;re just an engineer, but that doesn&#8217;t mean you might not someday build a geegaw or a thingamabob that may get you a thank you in someone else&#8217;s Nobel prize acceptance speech.</p>
<p>Raj: Is the trick making him disappear? Sure, let&#8217;s see it.</p>
<p>Howard: Here, shuffle these.</p>
<p>Raj: Okay.</p>
<p>Howard: Now spread them out on the table face down. Pick one, look at it.</p>
<p>Raj: Okay.</p>
<p>Howard: Now remember your card, put it back in the deck.</p>
<p>Sheldon: Is it any wonder he doesn&#8217;t have a doctorate?</p>
<p>Howard: Remind me, what&#8217;s your birthday?</p>
<p>Raj: October sixth.</p>
<p>Howard: Okay, October is the tenth month. So ten, one plus zero is one, plus six is seven. Turn over the seventh card.</p>
<p>Raj: How about that!</p>
<p>Howard: Is that your card?</p>
<p>Raj: Yes, it is. Very cool!</p>
<p>Sheldon: It&#8217;s not cool. It&#8217;s a childish trick designed to confuse and intrigue simpletons. How&#8217;d you do it?</p>
<p>Howard: A magician never reveals his secrets. But surely a future Nobel prize winner can figure it out.</p>
<p>Sheldon: Fine. Give me a second.</p>
<p>Howard: You ever notice when he thinks real hard, it smells like bacon?</p>
<p><em>(Leonard enters, bumps into random man)</em></p>
<p><em> </em></p>
<p>Random man: Oh, hey!</p>
<p>Leonard: Sorry! Oop!</p>
<p><em>(Continues across room bumping into nearly everyone in the room and knocking many things over)</em></p>
<p>Leonard: Sorry! Pardon me! Oh, I&#8217;m so sorry! My fault! Check it out. I just got contacts.</p>
<p><em>Credits sequence. </em></p>
<p><em> </em></p>
<p><em>Scene: The apartment.</em></p>
<p><em> </em></p>
<p>Penny: One, two, three, four, five. No way.</p>
<p>Howard: That&#8217;s your card, right?</p>
<p>Penny: Seven of clubs! That is amazing!</p>
<p>Sheldon: It&#8217;s not amazing. All magic tricks are merely a combination of manipulation and misdirection with the occasional dwarf in a box wiggling his feet.</p>
<p>Howard: Oh, really? So how did I do it?</p>
<p>Sheldon: Well, I don&#8217;t care how you did it. I have more important things to think about. Clearly, the cards are marked.</p>
<p>Howard: I thought you had more important things to think about.</p>
<p>Sheldon: I do. You just happen to have caught me on a break. Oh, let me see those cards.</p>
<p>Penny: Oh, big deal. Not knowing is part of the fun.</p>
<p>Sheldon: Not knowing is part of the fun. Was that the motto of your community college?</p>
<p>Leonard: Hey, guys, what do you think? Priya took me clothes shopping.</p>
<p>Penny: Hey, you look great. Good for you, Priya. I could never get that stupid hoodie off him when we were&#8230; well, you look great.</p>
<p>Leonard: Thank you.</p>
<p>Priya: Lets go hang everything up.</p>
<p>Leonard: Yeah, good. New contacts. I should have done this years ago.</p>
<p>Howard: So you finally getting used to them doing it on a daily basis?</p>
<p>Penny: I&#8217;m not going out with him. He can sleep with whoever he wants.</p>
<p>Howard: Yeah, I was talking to Raj.</p>
<p>Penny: Oopsy.</p>
<p>Sheldon: This deck is rigged in some fashion.</p>
<p>Howard: Fine. Get another deck and I&#8217;ll do the trick with that.</p>
<p>Sheldon: So you&#8217;re saying this is a regulation deck?</p>
<p>Howard: I&#8217;m saying believe in magic, you muggle.</p>
<p>Penny: You know, well, I&#8217;m happy Leonard&#8217;s found someone. Not that anybody asked.</p>
<p><em>Scene: Leonard’s bedroom</em></p>
<p>Priya: How many Star Trek uniforms do you have?</p>
<p>Leonard: Just two. Everyday and dress.</p>
<p>Priya: Uh-huh. Listen, we need to talk about something.</p>
<p>Leonard: Oh, my god, you&#8217;re breaking up with me! Why would you take me out shopping and then break up with me? That is so cruel.</p>
<p>Priya: Leonard, I&#8217;m not breaking up with you.</p>
<p>Leonard: Oh, okay. So, what&#8217;s up?</p>
<p>Priya: It&#8217;s a little weird, your ex-girlfriend hanging out here all the time.</p>
<p>Leonard: I know. I usually never see my ex-girlfriends, unless their hard drives crash.</p>
<p>Priya: I&#8217;m just saying, how would you feel if I was hanging out with my ex-boyfriend?</p>
<p>Leonard: Um, not great, I guess. But Penny and I are just friends.</p>
<p>Priya: I don&#8217;t care. This is a woman you have slept with. If you want her around, then I have to wonder if maybe you&#8217;re not ready to move on.</p>
<p>Leonard: No, no-no-no, I&#8217;m ready! I gave up the gift of sight for you. If that&#8217;s not moving on, what is?</p>
<p>Priya: You need to cut the cord with Penny.</p>
<p>Leonard: Oh. Okay.</p>
<p>Priya: You don&#8217;t sound very confident.</p>
<p>Leonard: No, no, I&#8217;m confident. It’s not like I&#8217;m unfamiliar with the get lost conversation. Although, this will be the first time I&#8217;ll be the one saying it.</p>
<p>Priya: Thank you. I appreciate it.</p>
<p>Leonard: Sure. Oh, god, you smell so good. Of course, that could just be my other senses getting stronger. Hypothetically, if I had access To a Lieutenant Uhura uniform, would you wear it?</p>
<p>Priya: Leonard, it is a source of great pain to me and my family that my brother has that outfit in his wardrobe. Can we not discuss it?</p>
<p>Leonard: I&#8217;m sorry. I’m, never mind.</p>
<p><em>Scene: The apartment</em></p>
<p>Howard: And was your card the jack of diamonds?</p>
<p>Penny: Oh, unbelievable! Know how he did it yet?</p>
<p>Sheldon: Um&#8230;</p>
<p>Penny: Aw, he has the same look my little nephew gets when he can&#8217;t figure out how I got his nose.</p>
<p>Howard: You know, I have some remedial magic tricks that might be more your speed. Like, ooh, look! The pencil&#8217;s rubber!</p>
<p>Penny: Hey, I think I know how you did the card trick.</p>
<p>Sheldon: Oh, oh, please, if I don&#8217;t know, you don&#8217;t know. That&#8217;s axiomatic.</p>
<p>Penny: Come here. <em>(Whispers to Howard)</em></p>
<p>Howard: You&#8217;re right.</p>
<p>Penny: Not too bad for someone who doesn&#8217;t know what axiomatic means.</p>
<p><em>Scene: The laundry room.</em></p>
<p>Leonard: Oh, hi.</p>
<p>Penny: Hey, you. What happened with the contacts?</p>
<p>Leonard: One of them&#8217;s upstairs. The other one sort of slipped back into my skull.</p>
<p>Penny: Hey, is that one of the new shirts Priya got you?</p>
<p>Leonard: Yeah.</p>
<p>Penny: You don&#8217;t wanna wash that.</p>
<p>Leonard: No?</p>
<p>Penny: No, that&#8217;s silk. Seriously, what would you do without me?</p>
<p>Leonard: Yeah.</p>
<p>Penny: You know, I gotta tell you, I think you got a real winner with Priya.</p>
<p>Leonard: Yeah, me, too.</p>
<p>Penny: She seems really sweet. Believe me, you do not want to take that for granted. There are a lot of bitches out there.</p>
<p>Leonard: Yeah, bitches are the worst.</p>
<p>Penny: All right, well I&#8217;ll see you later.</p>
<p>Leonard: Uh, Penny, there&#8217;s something I have to tell you.</p>
<p>Penny: What?</p>
<p>Leonard: Okay, uh, how do I put this? Um, are you familiar with Darwin&#8217;s observation of the finches in the Galapagos Islands?</p>
<p>Penny: Did they make a movie about it?</p>
<p>Leonard: No.</p>
<p>Penny: Then no.</p>
<p>Leonard: All right, well, anyway, Darwin observed that when two groups of finches competed over the same food source, eventually one of them would evolve a different beak shape so they could feed on something else.</p>
<p>Penny: Okay.</p>
<p>Leonard: So what do you think we can learn from that behaviour that we can apply to our own lives?</p>
<p>Penny: Uh, a bird in the hand is worth two in the bush?</p>
<p>Leonard: Sure, that has birds in it.</p>
<p>Penny: It&#8217;s fun talking to you, Leonard, I always learn stuff.</p>
<p>Leonard: I&#8217;m sorry, but I&#8217;m gonna count that.</p>
<p><em>Scene: Leonard’s bedroom</em></p>
<p>Priya: Do you really have to wear that in bed?</p>
<p>Leonard: Yeah. The doctor didn&#8217;t want me to rub my eye during the night. It was either this or one of those giant neck cones.</p>
<p>Priya: So did you get to talk to Penny?</p>
<p>Leonard: Oh, yes. Indeed, I did.</p>
<p>Priya: And?</p>
<p>Leonard: She completely understood everything I said. Case closed.</p>
<p>Priya: Was she upset?</p>
<p>Leonard: Maybe, but that&#8217;s not my problem. She&#8217;s not my girlfriend, you are.</p>
<p>Priya: I hope you weren&#8217;t cruel to her.</p>
<p>Leonard: Is the autumn cruel for letting the flowers die, or is that just nature&#8217;s way?</p>
<p>Sheldon: <em>(Knock, knock, knock) </em>Leonard? <em>(Knock, knock, knock) </em>Leonard? <em>(Knock, knock, knock) </em>Leonard?</p>
<p>Leonard: What?</p>
<p>Sheldon: I think I figured out Wolowitz&#8217;s magic trick, and I need you to pick a card.</p>
<p>Leonard: I am not opening that door, Sheldon.</p>
<p>Sheldon: As you wish. <em>(Cards slide under door) </em>Pick a card, put it back, and prepare to be amazed. <em>(Leonard does not) </em>Did you pick one?</p>
<p>Leonard: Yep.</p>
<p>Sheldon: <em>(Sliding card under door) </em>Is this your card?</p>
<p>Leonard: <em>(Not looking) </em>Nope.</p>
<p>Sheldon: Drat. Is this your card?</p>
<p>Leonard: No.</p>
<p>Sheldon: I&#8217;ll be right back.</p>
<p><em>Scene: The apartment. Sheldon is alone.</em></p>
<p>Sheldon: Pick a card, Sheldon. Now look at it, and put it back in the deck. Now do you remember your card? Of course I do, I have an eidetic memory. My apologies. Now shuffle the cards. Shuffling. Wait here. Processing image, cross-referencing.</p>
<p>Leonard: Whatcha doin&#8217;?</p>
<p>Sheldon: I&#8217;m reverse engineering Wolowitz&#8217;s magic trick.</p>
<p>Leonard: What&#8217;s up with the infrared cameras?</p>
<p>Sheldon: I&#8217;m measuring residual heat levels on the playing cards to determine which one&#8217;s been touched. By the way, if you hope to have children, I suggest you switch from briefs to boxers. Your testicles look a tad warm.</p>
<p>Leonard: Holy crap, are you connected to the Oak Ridge National Laboratory?</p>
<p>Sheldon: Yes. I&#8217;m using their Cray Supercomputer to analyze shuffling patterns.</p>
<p>Leonard: Sheldon, that computer is used for National defence. Hacking into it is a Federal crime.</p>
<p>Sheldon: Relax, we&#8217;re not under attack right now.</p>
<p>Leonard: Okay, I&#8217;m leaving before the black helicopters get here.</p>
<p>Sheldon: And is this my card? Rats! I wonder if Howard used a radioactive tracer. Where am I going to find Uranium-235 this time of night? Come on, Craigslist.</p>
<p><em>Scene: A restaurant</em></p>
<p>Leonard: Is it racist that I took you to an Indian restaurant?</p>
<p>Priya: It&#8217;s okay, I like Indian food.</p>
<p>Leonard: Or as you probably call it back home, food.</p>
<p>Priya: Why didn&#8217;t you wear your contacts? Don&#8217;t you like them?</p>
<p>Leonard: No, no, I do. It&#8217;s just I knew I was having curry tonight and I only want one part of my body to burn at a time.</p>
<p>Priya: You know, my company offered me a position here in Los Angeles.</p>
<p>Leonard: Does that mean you&#8217;d relocate here permanently?</p>
<p>Priya: Maybe. What do you think?</p>
<p>Leonard:Oh,  I love it. Oh, that&#8217;s great!</p>
<p>Penny <em>(knocking on window)</em>: Hi! Hi! Just hang on.</p>
<p>Leonard: Small world.</p>
<p>Priya: You did talk to her, didn&#8217;t you?</p>
<p>Leonard: Of course, I did. I talked and talked and talked.</p>
<p>Penny: Hey! Look at Leonard in his fancy-ass grownup clothes. Way to go, Priya. For once, he doesn&#8217;t look like a mannequin in the boys&#8217; department. Hey, did he tell you I saved the silk shirt?</p>
<p>Priya: No, he did not.</p>
<p>Penny: He was gonna throw it in the washing machine with his Spiderman underwear. That&#8217;s our Lenny, huh?</p>
<p>Priya: Yes, that&#8217;s our Lenny.</p>
<p>Penny: So, how&#8217;s the food here? Oh, wow, that is really, really good. Oh, I&#8217;m sorry. You guys are on a date. We can hang out anytime. Have fun.</p>
<p>Leonard: Bye.</p>
<p>Priya: Good night.</p>
<p>Penny: Oh, that&#8217;s, hot, hot, hot, hot! Whoo! It really sneaks up on ya, huh? All right, I gotta go.</p>
<p>Leonard: It is pretty spicy.</p>
<p><em>Scene: The cafeteria.</em></p>
<p>Sheldon: Oh, I&#8217;ve been meaning to tell you, I figured out your magic trick.</p>
<p>Howard: Really?</p>
<p>Sheldon: Oh, yes. In fact, I improved upon it.</p>
<p>Howard: I&#8217;d love to see that. How about you, Raj?</p>
<p>Raj: Beefaroni and a show? How do you turn that down?</p>
<p>Sheldon: All right. Pick a card.</p>
<p>Howard: Okay.</p>
<p>Sheldon: Bippity-boppity-boo.</p>
<p>Howard: What&#8217;s with the wand and the beep?</p>
<p>Sheldon: The wand is called showmanship, and the beep is none of your business. Oh! Excuse me, I&#8217;m getting a text message completely unrelated to this magic trick. Oh, look, my dry cleaning&#8217;s ready. And your card was the five of spades. Ta-da.</p>
<p>Raj: These cards have barcodes on them. The wand is a reader. It&#8217;s transmitting to your phone.</p>
<p>Sheldon: I said, ta-da. Show&#8217;s over.</p>
<p>Howard: That&#8217;s pathetic. Let me show you how a real magician does it. Raj, take a card. Don&#8217;t let me see it.</p>
<p>Raj: Okay.</p>
<p>Howard: Three of clubs. <em>(It isn’t)</em></p>
<p>Raj: Son of a gun, you&#8217;re blowing my mind!</p>
<p>Howard: Bippity-boppity-boo-yah!</p>
<p>Sheldon: That does it. I&#8217;m getting uranium.</p>
<p>Raj: You ever gonna tell him?</p>
<p>Howard: Maybe. When it stops being fun.</p>
<p>Raj: So never.</p>
<p><em>Scene: Penny’s door.</em></p>
<p>Penny: Oh, hey, I was just on my way to work.</p>
<p>Leonard: Oh, okay, it’s not important. It can wait.</p>
<p>Penny: No, it&#8217;s all right. Walk me down. So, what&#8217;s up?</p>
<p>Leonard: Uh, I kind of have a problem I was hoping you could help me with.</p>
<p>Penny: Sure, anything.</p>
<p>Leonard: Move out.</p>
<p>Penny: What?</p>
<p>Leonard: Well, uh, not far. Hey, if you ever wanna start a family, La Cañada has some great schools.</p>
<p>Penny: Okay, I&#8217;m not moving anywhere. What the hell is this all about?</p>
<p>Leonard: Why does it have to be about anything? Can&#8217;t a fella ask his buddy to relocate, no questions asked?</p>
<p>Penny: Oh, for god&#8217;s sake, Leonard, this is about Priya, isn&#8217;t it? She doesn&#8217;t want me hanging out with you.</p>
<p>Leonard: Yes. There, I said it.</p>
<p>Penny: Okay, look, I happen to like your girlfriend.</p>
<p>Leonard: And she likes you.</p>
<p>Penny: No, she doesn&#8217;t.</p>
<p>Leonard: Not really, no.</p>
<p>Penny: It doesn&#8217;t matter. Look, I promise from now on I will keep my distance from you.</p>
<p>Leonard: Well, now, hold on. What kind of distance are we talking about? Because we are neighbours. I mean, I can hear the toilet flush in your apartment.</p>
<p>Penny: You can hear my toilet flush?</p>
<p>Leonard: I don&#8217;t listen for it, but it&#8217;s nice to know everything&#8217;s okay with your plumbing. The building&#8217;s plumbing.</p>
<p>Penny: Leonard, I get it. You&#8217;re in a new relationship now. And I&#8217;m happy for you. So why don&#8217;t we just shake hands and part friends?</p>
<p>Leonard: Well, now hold on, how about this? How about we still hang out, but on the down-low?</p>
<p>Penny: Are you really that kind of guy?</p>
<p>Leonard: No. I actually felt kinda silly just saying on the down-low.</p>
<p>Penny: Good-bye, Leonard.</p>
<p>Leonard: Penny, wait. <em>(Bumps into door) </em>Damned contacts.</p>
<p><em>Scene: The cafeteria.</em></p>
<p>Howard: Incoming.</p>
<p>Raj: Hi!</p>
<p>Howard: Hey!</p>
<p>Sheldon: Hello.</p>
<p>Raj: Why so glum, chum?</p>

<p>Howard: Oh, that&#8217;s too bad. Figure out the magic trick yet?</p>
<p>Sheldon: Figure out the magic trick yet?</p>
<p>Howard: Want me to tell you how to do it?</p>
<p>Sheldon: No.</p>
<p>Howard: I’ll show you one more time. Raj?</p>
<p>Sheldon: Hang on. This time do it with me, so I can make sure there&#8217;s no monkey business.</p>
<p>Howard: All right. <em>(Raj looks over Sheldon’s shoulder. Holds two fingers over his heart). </em>Two of hearts.</p>
<p>Sheldon: I hate you.</p>
<p>Howard: Yeah, he&#8217;s gonna win the Nobel prize.</p>

			function JetpackLikesWidgetQueueHandler() {

					setTimeout( JetpackLikesWidgetQueueHandler, 500 );
					return;
				}

				if ( jetpackLikesWidgetQueue.length > 0 ) {
					// We may have a widget that needs creating now
					var found = false;
					while( jetpackLikesWidgetQueue.length > 0 ) {
						// Grab the first member of the queue that isn't already loading.
						wrapperID = jetpackLikesWidgetQueue.splice( 0, 1 )[0];
						if ( jQuery( '#' + wrapperID ).hasClass( 'jetpack-likes-widget-unloaded' ) ) {
							found = true;
							break;
						}
					}
					if ( ! found ) {
						setTimeout( JetpackLikesWidgetQueueHandler, 500 );
						return;
					}
				} else if ( jQuery( 'div.jetpack-likes-widget-unloaded' ).length > 0 ) {
					// Grab any unloaded widgets for a batch request
					JetpackLikesBatchHandler();

					// Get the next unloaded widget
					wrapperID = jQuery( 'div.jetpack-likes-widget-unloaded' ).first()[0].id;
					if ( ! wrapperID ) {
						// Everything is currently loaded
						setTimeout( JetpackLikesWidgetQueueHandler, 500 );
						return;
					}
				}

				if ( 'undefined' === typeof wrapperID ) {
					setTimeout( JetpackLikesWidgetQueueHandler, 500 );
					return;
				}

				var $wrapper = jQuery( '#' + wrapperID );
				$wrapper.find( 'iframe' ).remove();

				if ( $wrapper.hasClass( 'slim-likes-widget' ) ) {
					$wrapper.find( '.post-likes-widget-placeholder' ).after( "<iframe class='post-likes-widget jetpack-likes-widget' name='" + $wrapper.data( 'name' ) + "' height='22px' width='68px' frameBorder='0' scrolling='no' src='" + $wrapper.data( 'src' ) + "'></iframe>" );

				}

				$wrapper.removeClass( 'jetpack-likes-widget-unloaded' ).addClass( 'jetpack-likes-widget-loading' );

				$wrapper.find( 'iframe' ).load( function( e ) {
					var $iframe = jQuery( e.target );

						$wrapper.find( 'iframe' ).Jetpack( 'resizeable' );
					}
				});
