		<h2 class="title">Series 7 Episode 18 &#8211; The Mommy&nbsp;Observation</h2>
			<div class="entrytext">
				<p><i>Scene: The apartment.</i></p>
<p>Amy: Should you really be sitting in Sheldon&#8217;s spot?</p>
<p>Raj: He&#8217;s in Texas, he&#8217;ll never know.</p>
<p>Penny: I wouldn&#8217;t be so sure about that.</p>
<p>Leonard: Yeah, he has a very sensitive butt. Well, it&#8217;s true. Once I saw him sit on a bunch of loose change and add it up.</p>
<p>Raj: Have you heard from Howard?</p>
<p>Bernadette: I did. His talk at NASA went great.</p>
<p>Penny: Sheldon didn&#8217;t heckle him?</p>
<p>Bernadette: No, in fact, he was so well-behaved, Howie bought him a Buzz Aldrin bobble head and astronaut ice cream.</p>
<p>Stuart: Hey, guys.</p>
<p>Amy: Oh, hi.</p>
<p>Bernadette: Hi, Stuart.</p>
<p>Raj: Hey, Stuart!</p>
<p>Stuart: How&#8217;s it going?</p>
<p>Raj: Good. Sheldon&#8217;s out of town, so we can do whatever we want. We even ordered from the Thai place he doesn&#8217;t like.</p>
<p>Stuart: Oh, how is it?</p>
<p>Penny: Disgusting. Do not tell him.</p>
<p>Leonard: What do you guys want to do tonight?</p>
<p>Amy: I don&#8217;t know.</p>
<p>Bernadette: Well, I told Howie if I wasn&#8217;t busy, I&#8217;d spend the night at his mom&#8217;s. So for God&#8217;s sake, think of something.</p>
<p>Raj: Stuart? Are you okay?</p>
<p>Stuart: No, I don&#8217;t feel so&#8230;</p>
<p>Leonard: Oh, my God, Stuart?</p>
<p>Penny: Oh, my God, you guys need to do something!</p>
<p>Raj: Stuart?</p>
<p>Leonard: Calling 911.</p>
<p>Raj: Well, it&#8217;s too late.</p>
<p>Leonard: What do you mean it&#8217;s too late?</p>
<p>Raj: He&#8217;s been murdered by someone in this room.</p>
<p>Penny: Oh, my God.</p>
<p>Leonard: Oh, come on.</p>
<p>Raj: Welcome to another classic Koothrappali murder mystery dinner.</p>
<p>Amy: I&#8217;m leaving.</p>
<p>Raj: You can&#8217;t leave. You&#8217;re a suspect in the mysterious murder of Stuart Bloom.</p>
<p>Bernadette: I didn&#8217;t know his last name was Bloom.</p>
<p>Amy: Yeah, it&#8217;s Bloom.</p>
<p>Penny: Are you really gonna lie on the floor and pretend to be dead all night?</p>
<p>Stuart: What do you think I was gonna do at home?</p>
<p><i>Credits sequence.</i></p>
<p><i>Scene: The apartment.</i></p>
<p>Penny: Are you gonna make us pretend to be a bunch of lame characters with silly accents?</p>
<p>Raj: Lame characters with silly ac&#8230;, what kind of actress are you?</p>
<p>Penny: You&#8217;re right, I&#8217;m sorry. Sounds like fun.</p>
<p>Raj: Thank you.</p>
<p>Penny: That kind of actress. This sucks.</p>
<p>Leonard: Come on, guys. Raj put a lot of effort into this. And that&#8217;s great. It&#8217;s not sad, it&#8217;s great.</p>
<p>Amy: I think it might be fun to be someone else tonight.</p>
<p>Raj: Actually, you&#8217;re all just gonna be yourselves.</p>
<p>Amy: Oh, ugh.</p>
<p>Bernadette: All right, if I&#8217;m doing this, I&#8217;m playing to win, so just to be clear, if we&#8217;re ourselves, that means one of us killed Stuart?</p>
<p>Raj: Very good, Bernadette. You are a regular Byomkesh Bakshi.</p>
<p>Bernadette: What is that, like, the Indian Sherlock Holmes?</p>
<p>Raj: Or is Sherlock Holmes the English Byomkesh Bakshi?</p>
<p>Amy: According to Wikipedia, Sherlock Holmes came first.</p>
<p>Raj: Great, everyone&#8217;s a Byomkesh Bakshi. Now, here are some secret facts about each of you, including whether you are the murderer. Throughout the game, feel free to ask each other questions to uncover clues.</p>
<p>Penny: Got it. Hey, who&#8217;s the murderer?</p>
<p>Raj: Any question but that.</p>
<p>Penny: Sorry. Hey, who&#8217;s not the murderer?</p>
<p>Stuart: Bernadette, can you not stand so close to me?</p>
<p>Bernadette: What do you care? You&#8217;re dead.</p>
<p>Stuart: Suit yourself. But I can kind of see up your skirt.</p>
<p><i>Scene: A hire car.</i></p>
<p>Sheldon: Thank you for my bobble head.</p>
<p>Howard: Thank you for not making fun of me during my speech.</p>
<p>Sheldon: It wasn&#8217;t easy. Was it Buzz?</p>
<p>Howard: We can&#8217;t show up to your mom&#8217;s empty-handed. We should bring something.</p>
<p>Sheldon: I already am. I&#8217;m bringing the gift of knowledge.</p>
<p>Howard: Oh, boy.</p>
<p>Sheldon: Despite what her Bible says, recent archaeological studies indicate that although camels are referenced in the Old Testament, they didn&#8217;t exist in the Middle East until hundreds of years later.</p>
<p>Howard: I was gonna say we pick up a cake or a pie. But an insult to her faith is always thoughtful.</p>
<p><i>Scene: The apartment. </i></p>
<p>Bernadette: So, what happens next?</p>
<p>Raj: I can&#8217;t tell you that. But perhaps the killer dropped a fun and imaginative clue somewhere in the apartment.</p>
<p>Penny: Ooh, I&#8217;m gonna check the fridge, and see if there are any clues inside a beer.</p>
<p>Amy: Hey, I found something. It looks like a little man with a briefcase.</p>
<p>Leonard: Oh, no, no. That&#8217;s Clarence Darrow. It&#8217;s from a game Sheldon made up called Chutes and Lawyers. You slide down a chute and then work your way back up through the appellate system.</p>
<p>Raj: Well, unlike that, my games are much more fun.</p>
<p>Penny: Okay, can I ask you something? Why do you like making us do this stuff?</p>
<p>Raj: Well, I guess it goes back to when I was a fat kid in India, and didn&#8217;t have any friends.</p>
<p>Bernadette: I didn&#8217;t know you were fat.</p>
<p>Raj: Yeah, I was. I was 200 pounds by the time I was in middle school. Kids were mean. Cows may be sacred there, but it doesn&#8217;t help if you look like one. Anyway, I was pretty lonely, so I had to make up my own stories and games, and I promised myself if I ever made any friends that, that I would play those games with them.</p>
<p>Amy: That is so sweet.</p>
<p>Leonard: I&#8217;ve seen old pictures of you. You were never a fat kid.</p>
<p>Raj: No, I was svelte as a gazelle. A gazelle blessed with a flair for storytelling.</p>
<p>Bernadette: Hey, what about this? It looks like a receipt.</p>
<p>Amy: What&#8217;s it for?</p>
<p>Bernadette: For a cup of coffee. But it&#8217;s dated 20 years from now.</p>
<p>Raj: From the future? How fun and imaginative.</p>
<p>Leonard: So one of us came back from the future to murder Stuart?</p>
<p>Raj: Correct.</p>
<p>Leonard: Does the gazelle with a flair for storytelling know he&#8217;s just ripping off Terminator?</p>
<p>Raj: Does the smart-ass know that Terminator was actually ripped off from an Outer Limits script called Demon With a Glass Hand, by Harlan Ellison?</p>
<p>Leonard: Oh, does the gazelle know that according to Harlan Ellison, it was not ripped off from Demon With a Glass Hand, but was ripped off from another Outer Limits script he wrote called The Soldier?</p>
<p>Penny: I&#8217;m gonna need another clue.</p>
<p><i>Scene: Sheldon&#8217;s childhood home.</i></p>
<p>Howard: Okay, give me the flowers and pie.</p>
<p>Sheldon: But if we show up and you&#8217;re holding them, she&#8217;ll think they&#8217;re only from you.</p>
<p>Howard: They are only from me. You said the gift of you was enough.</p>
<p>Sheldon: Yes, but now that I&#8217;ve seen what the gift of me with flowers and pie looks like, there&#8217;s no going back. Boy, I can&#8217;t wait to see the look on her face. We&#8217;re leaving right now.</p>
<p>Howard: What&#8217;s wrong?</p>
<p>Sheldon: Nothing.</p>
<p>Howard: What? Sheldon, tell me what&#8217;s going on.</p>
<p>Sheldon: I saw my mommy with a naked man and she was trying to be a mommy again.</p>
<p><i>Scene: A bar in Texas.</i></p>
<p>Howard: You ready to talk about it?</p>
<p>Sheldon: I&#8217;m not nearly drunk enough. Okay. Do you have any idea what it&#8217;s like to see your mother ravaging someone?</p>
<p>Howard: Does a brisket count? I&#8217;m sorry, buddy, that&#8217;s rough. But didn&#8217;t she know we were coming?</p>
<p>Sheldon: No. I wanted to surprise her. What am I supposed to say to her after something like this?</p>
<p>Howard: Well, did she see you in the window?</p>
<p>Sheldon: No.</p>
<p>Howard: Then don&#8217;t say anything.</p>
<p>Sheldon: I have to. How can we ever hope to have a healthy relationship if I don&#8217;t tell her how disappointed I am, and that I&#8217;ll never forgive her?</p>
<p>Howard: Well, don&#8217;t do it on the phone.</p>
<p>Sheldon: No, I&#8217;m just going to tell her I&#8217;m coming so she can give that good time Charlie the heave-ho.</p>
<p>Howard: That&#8217;s some tough talk, I may need to cut you off.</p>
<p>Sheldon: Yeah, maybe you better, I took a pretty big sip.</p>
<p>Howard: Look, she&#8217;s a grown woman. And, and your dad&#8217;s been gone a long time. Maybe this is none of your business.</p>
<p>Sheldon: I beg to differ. I used to live in those genitals. And if someone wants to move into my old room, I should at least get a vote.</p>
<p><i>Scene: The apartment.</i></p>
<p>Raj: Okay, murder suspects, Leonard has found the time machine the killer must&#8217;ve used. You&#8217;re all inside it, and we hurtle through the very fabric of time. And welcome to 20 years in the future. Hey, you guys just time-travelled. Stop looking so bored.</p>
<p>Penny: Well, my beer isn&#8217;t flat and my rack&#8217;s not saggy. So far, the future&#8217;s great.</p>
<p>Stuart: Hey, can I go to the bathroom?</p>
<p>Raj: Fine, just try not to look too alive.</p>
<p>Stuart: That&#8217;s my jam.</p>
<p>Bernadette: So one of us went back in time to kill Stuart?</p>
<p>Amy: But why?</p>
<p>Raj: Perhaps this will help. Here are some facts about yourselves in the future that might contain a clue.</p>
<p>Amy: Hey, I won the Nobel Prize in physiology, then I used the money to buy Stuart&#8217;s comic book store, and close it down so Sheldon would pay attention to me. Not the worst idea.</p>
<p>Penny: Hey, I&#8217;m a famous actress living in London.</p>
<p>Leonard: Hmm, I&#8217;m a professor at Stanford.</p>
<p>Penny: Hmm.</p>
<p>Bernadette: So I guess you two are making it work long distance.</p>
<p>Raj: Oh, no. In this game, as your careers both took off, you drifted apart. Kind of like how future me lost touch with you guys after I became boy toy for the wrinkled, but still flexible, Madonna.</p>
<p>Amy: What does your card say, Bernadette?</p>
<p>Leonard: Uh, uh, uh, hang on. Do you think we&#8217;d really drift apart if we both became successful?</p>
<p>Penny: Of course not. If I became a famous actress and had to move, you would just come with me.</p>
<p>Leonard: But if I get a chance to be a tenured professor, I might not have that much choice in where I end up.</p>
<p>Penny: Yeah, but if I become a successful actress, we wouldn&#8217;t need the money.</p>
<p>Leonard: Well, you don&#8217;t go into science for the money.</p>
<p>Bernadette: Speak for yourself. Last month, my company both invented and cured Restless Eye Syndrome. Ka-ching, ya blinky chumps.</p>
<p>Penny: So, wait, if my career took me somewhere else, you might not go?</p>
<p>Amy: Dun, dun, dun. He started it.</p>
<p>Raj: Okay, okay, guys, new back stories. Uh, Penny and Leonard, you two lovebirds are super successful and living in New York. Uh, you&#8217;re an actress, you&#8217;re a professor and you have three beautiful kids.</p>
<p>Leonard: Great.</p>
<p>Penny: Really? You think I&#8217;m putting this body through three kids?</p>
<p><i>Scene: Sheldon&#8217;s childhood home.</i></p>
<p>Sheldon: Okay, let&#8217;s do this.</p>
<p>Howard: Good luck.</p>
<p>Sheldon: You, aren&#8217;t you gonna come with me?</p>
<p>Howard: While you confront your mother about her sex life? I&#8217;d rather go back to that bar in ass-less chaps.</p>
<p>Sheldon: <i>(Knock, knock, knock) </i>Mother? <i>(Knock, knock, knock) </i>Mother? <i>(Knock, knock, knock) </i>Mother?</p>
<p>Mrs Cooper: Shelly, I&#8217;m so glad you&#8217;re here.</p>
<p>Sheldon: I saw you having naked sex.</p>
<p>Mrs Cooper: What are you talking about?</p>
<p>Sheldon: Earlier, I came here to surprise you. I looked in the window and I saw you with a man.</p>
<p>Mrs Cooper: Oh, Shelly, I&#8217;m so sorry. Come in. Um, maybe we should sit down and talk about this.</p>
<p>Sheldon: Can you recommend a surface you haven&#8217;t had coitus on?</p>
<p>Mrs Cooper: That&#8217;s not funny. Maybe we should sit at the table. Well, I&#8217;m sure that, uh, you have a lot of questions.</p>
<p>Sheldon: Who was he?</p>
<p>Mrs Cooper: His name is Ron. I met him at my prayer group.</p>
<p>Sheldon: How long have you been involved with him?</p>
<p>Mrs Cooper: A few months.</p>
<p>Sheldon: And of those few months, how long have you been a demented sex pervert?</p>
<p>Mrs Cooper: That is no way to speak to your mother.</p>
<p>Sheldon: Perhaps not. But it is a way to speak to a woman who quoted the Bible to me my whole life, and then desecrates one of Ethan Allen&#8217;s finest sofas.</p>
<p>Mrs Cooper: I will give you one opportunity, young man, to apologize.</p>
<p>Sheldon: Or what?</p>
<p>Mrs Cooper: Or I will send you to your room.</p>
<p>Sheldon: That&#8217;s ridiculous. I am a grown man. I am a professional scientist. And I currently occupy the moral high ground.</p>
<p>Mrs Cooper: Go to your room.</p>
<p>Sheldon: But I occupy the moral high ground.</p>
<p>Mrs Cooper: Go to your room.</p>
<p>Sheldon: But I&#8217;m a professional scientist.</p>
<p>Mrs: Go to your room!</p>
<p>Sheldon: I&#8217;m a grown man.</p>
<p><i>Scene: The apartment.</i></p>
<p>Raj: Can we please get back to the game?</p>
<p>Leonard: In a minute. I don&#8217;t understand why any success you have in acting is more important than any I have in science.</p>
<p>Penny: Okay, if you do something cool in science, you might change the world. If I become a famous actress, I&#8217;m not gonna tell you why movie stars are the best. They just are.</p>
<p>Amy: I&#8217;m surprised you guys never talked about this stuff.</p>
<p>Leonard: Like you and Sheldon have everything figured out?</p>
<p>Amy: Actually, our relationship agreement covers a wide array of scenarios, including career changes, financial instability, intelligent dog uprising. FYI, we plan on selling out the human race hard.</p>
<p>Penny: In 20 years, who knows what&#8217;ll happen with any of us?</p>
<p>Stuart: I think you and Leonard will be together.</p>
<p>Penny: You do?</p>
<p>Stuart: Yeah. I think you&#8217;re the best couple I know.</p>
<p>Leonard: Aw.</p>
<p>Penny: That&#8217;s so sweet.</p>
<p>Bernadette: What the hell?</p>
<p>Amy: Excuse me?</p>
<p>Penny: Ah-da-da-da-da, let the dead man talk. So, why do you think that?</p>
<p>Stuart: Uh, well, I feel like you guys make each other better. Penny brought Leonard out of his shell. And it seems like Leonard makes Penny think more deeply about the world. I don&#8217;t know. Together, you two kind of make one awesome person.</p>
<p>Penny: Aw, Stuart. Now I feel bad for murdering you.</p>
<p>Raj: Oh, come on.</p>
<p>Bernadette: Penny did it, I win. Suck it, jackasses.</p>
<p><i>Scene: Sheldon&#8217;s old bedroom.</i></p>
<p>Howard: Hey, can I come in?</p>
<p>Sheldon: Apparently any man is welcome in this house. Why not you? I thought you were waiting in the car.</p>
<p>Howard: That was an hour ago, Sheldon. A Jew sits in front of a house in Texas that long, For Sale signs start to go up. I take it things didn&#8217;t go smoothly with your mom.</p>
<p>Sheldon: No. I&#8217;m not sure how we&#8217;re going to get past this.</p>
<p>Howard: You know, I, I sort of went through something like this myself.</p>
<p>Sheldon: Howard, we&#8217;ve all seen your mother naked. That woman needs to learn how to tie a robe.</p>
<p>Howard: I&#8217;m talking about when my mom started seeing someone a couple of years after my dad left.</p>
<p>Sheldon: I didn&#8217;t know she did.</p>
<p>Howard: Yeah, she was dating this guy, and I was kind of a jerk to her about it.</p>
<p>Sheldon: What did you do?</p>
<p>Howard: Well, let&#8217;s just say it was the most vicious bar mitzvah speech in the history of Temple Beth El. Anyway, she broke up with him. And she&#8217;s basically been alone ever since. She never said it, but I always felt I was the reason why.</p>
<p>Sheldon: I&#8217;m sorry. Although, based on your story, you absolutely were the reason why.</p>
<p>Howard: All I&#8217;m saying is you might not want to get in the way of your mom&#8217;s happiness.</p>
<p>Sheldon: You may have a point. I love my mother. Even if she fornicates like a demonic weasel.</p>
<p>Howard: And the healing begins.</p>
<p><i>Scene: Mrs Cooper&#8217;s living room.</i></p>
<p>Sheldon: Hello.</p>
<p>Mrs Cooper: Hello. Are you ready to discuss this calmly, like adults?</p>
<p>Sheldon: I am.</p>
<p>Mrs Cooper: Shelly, I&#8217;m sorry that you saw what you saw. I know that this is hard for you.</p>
<p>Sheldon: I think what most upsets me about it, Mother, is the hypocrisy. Doesn&#8217;t this contradict all the religious rules you&#8217;ve been espousing your whole life?</p>
<p>Mrs Cooper: You&#8217;re right, it does, and it&#8217;s something that I&#8217;m struggling with these days.</p>
<p>Sheldon: Then why are you doing it?</p>
<p>Mrs Cooper: Because I&#8217;m not perfect, Shelly. And that man&#8217;s booty is.</p>
<p>Sheldon: Well, this is confusing for me. But I don&#8217;t want to stand in the way of your happiness. So, I&#8217;ll condemn you internally while maintaining an outward appearance of acceptance.</p>
<p>Mrs Cooper: That is very Christian of you.</p>
<p>Sheldon: Mother, if you&#8217;re going to conduct your life in this fashion, then I should let you know that the world has changed since you were a young woman. You know, it&#8217;s not all sock hops, soda jerks and segregation anymore.</p>
<p>Mrs Cooper: How old do you think I am?</p>
<p>Sheldon: My point is that you&#8217;re going to need to be careful. You used to be protected by your pre-Enlightenment mythology, but now you&#8217;re going to need a stronger friend named latex.</p>
<p>Mrs Cooper: Are you having the sex talk with me?</p>
<p>Sheldon: Well, someone has to.</p>
<p>Mrs Cooper: Oh, dear Lord.</p>
<p>Sheldon: No, don&#8217;t look to Him. He&#8217;s mad at you right now.</p>
<p><i>Scene: The apartment.</i></p>
<p>Penny: Oh, come on, don&#8217;t pout. I&#8217;m sorry I ruined your game.</p>
<p>Raj: I&#8217;m not pouting, I&#8217;m brooding. Which is how sexy men pout.</p>
<p>Amy: It actually was kind of fun.</p>
<p>Raj: You&#8217;re just saying that.</p>
<p>Amy: Yeah.</p>
<p>Bernadette: I liked the time travel element.</p>
<p>Raj: Thank you. I thought it was inspired.</p>
<p>Leonard: It was. By Terminator.</p>
<p>Bernadette: Well, I hope 20 years from now, we really are all still together and still friends.</p>
<p>Penny: Definitely.</p>
<p>Leonard: Of course. Well, hey, how about this? Whether we&#8217;re friends, not friends, scattered around the world, no matter what, let&#8217;s all promise to meet in front of this building exactly 20 years from tonight at 8:00 p.m. and have dinner like we always do.</p>
<p>Bernadette: Aw.</p>
<p>Penny: I love it.</p>
<p>Leonard: Putting it in my phone right now.</p>
<p>Penny: Yeah, me, too.</p>
<p>Stuart: My phone doesn&#8217;t have a calendar. So I&#8217;ll just write it on my hand.</p>
<p>Raj: And it&#8217;s done.</p>
<p>Penny: Yup, we&#8217;re all in.</p>
<p>Leonard: All right, I&#8217;ll see you guys in 20 years.</p>
<p><i>Scene: In front of the building, 20 years later.</i></p>
<p>Stuart: I knew it.</p>

			function JetpackLikesWidgetQueueHandler() {
				var wrapperID;
				if ( ! jetpackLikesMasterReady ) {
					setTimeout( JetpackLikesWidgetQueueHandler, 500 );
					return;
				}

				if ( jetpackLikesWidgetQueue.length > 0 ) {
					// We may have a widget that needs creating now
					var found = false;
					while( jetpackLikesWidgetQueue.length > 0 ) {
						// Grab the first member of the queue that isn't already loading.
						wrapperID = jetpackLikesWidgetQueue.splice( 0, 1 )[0];
						if ( jQuery( '#' + wrapperID ).hasClass( 'jetpack-likes-widget-unloaded' ) ) {
							found = true;
							break;
						}
					}
					if ( ! found ) {
						setTimeout( JetpackLikesWidgetQueueHandler, 500 );
						return;
					}
				} else if ( jQuery( 'div.jetpack-likes-widget-unloaded' ).length > 0 ) {
					// Grab any unloaded widgets for a batch request
					JetpackLikesBatchHandler();

					// Get the next unloaded widget
					wrapperID = jQuery( 'div.jetpack-likes-widget-unloaded' ).first()[0].id;
					if ( ! wrapperID ) {
						// Everything is currently loaded
						setTimeout( JetpackLikesWidgetQueueHandler, 500 );
						return;
					}
				}

				if ( 'undefined' === typeof wrapperID ) {
					setTimeout( JetpackLikesWidgetQueueHandler, 500 );
					return;
				}

				var $wrapper = jQuery( '#' + wrapperID );
				$wrapper.find( 'iframe' ).remove();

				if ( $wrapper.hasClass( 'slim-likes-widget' ) ) {
					$wrapper.find( '.post-likes-widget-placeholder' ).after( "<iframe class='post-likes-widget jetpack-likes-widget' name='" + $wrapper.data( 'name' ) + "' height='22px' width='68px' frameBorder='0' scrolling='no' src='" + $wrapper.data( 'src' ) + "'></iframe>" );

				}

				$wrapper.removeClass( 'jetpack-likes-widget-unloaded' ).addClass( 'jetpack-likes-widget-loading' );

				$wrapper.find( 'iframe' ).load( function( e ) {
					var $iframe = jQuery( e.target );
					$wrapper.removeClass( 'jetpack-likes-widget-loading' ).addClass( 'jetpack-likes-widget-loaded' );

						$wrapper.find( 'iframe' ).Jetpack( 'resizeable' );
					}
				});
				setTimeout( JetpackLikesWidgetQueueHandler, 250 );
