<p>Penny <em>(pouring out a shot of tequila)</em>: Would you say that&#8217;s about 40 milliliters?</p>
<p>Sheldon: More or less.</p>
<p>Penny: Great. <em>(Drinks the tequila) </em>Now, where were we?</p>
<p>Sheldon: I believe you were about to ask me to choose a cocktail. Fortunately, thanks to computer-savvy alcoholics, there&#8217;s an app for that. Let&#8217;s see. Harvey Wallbanger. Eh. Sex on the Beach. I hardly think so. Rob Roy, Silk Slipper, Mad Hatter. Ooh! I&#8217;ll have a Rosewater Ricky.</p>
<p>Penny: A what?</p>
<p>Sheldon: You&#8217;ll need pitted, brandied cherries, gin, rosewater, angostura bitters and overproofed rum. Now, first, dust the cherries with sugar, then spray them with a mixture of rum and bitters. Then ignite the rum, caramel&#8230;</p>
<p>Penny <em>(pouring out a shot of tequila)</em>: Here you go, one Rosewater Ricky.</p>
<p>Sheldon: That&#8217;s not how it looks in the picture.</p>
<p>Penny: Oh, yeah, you know, every bartender makes it differently. Why don&#8217;t you give it a try.</p>
<p>Sheldon: All right. To the metric system. <em>(Knocks back drink. Spits it all back into glass.) </em>I can&#8217;t taste the cherries.</p>
<p>Penny: All right. Sheldon, what&#8217;s on your mind?</p>
<p>Sheldon: I wronged Howard, and he won&#8217;t accept my apology.</p>
<p>Penny: Oh, right, that. Yes, Leonard told me. Sorry, honey.</p>
<p>Sheldon: Penny, you face failure on a daily basis. How do you cope?</p>
<p>Penny: I drink.</p>
<p>Sheldon: To drinking. <em>(Knocks back drink and spits it out again.) </em>Nope. Oh, if only there were some way to force Howard to accept my apology so I could escape this miasma of guilt.</p>
<p>Penny: You know, sometimes stuff just happens, and there&#8217;s nothing you can do about it. For example, Lisa Peterson hasn&#8217;t talked to me since the 11th grade, because no matter how much you apologize, you can&#8217;t go back and un-dry-hump someone&#8217;s boyfriend.</p>
<p>Sheldon: I see. You&#8217;re saying I&#8217;m facing Starfleet Academy&#8217;s unwinnable command scenario, the Kobayashi Maru.</p>
<p>Penny: Exactly. Sometimes you can&#8217;t win.</p>
<p>Sheldon: Captain Kirk won.</p>
<p>Penny: Kirk cheated.</p>
<p>Sheldon: Impressive that you know that. It&#8217;s hard to believe I&#8217;m actually having this conversation with you.</p>
<p>Penny: Right there with you.</p>
<p>Sheldon: Kirk beat the Kobayashi Maru by reprogramming the simulator. That&#8217;s it.</p>
<p>Penny: What?</p>
<p>Sheldon: I&#8217;ll reprogram Howard.</p>
<p>Penny: What? Sheldon, you can&#8217;t reprogram people.</p>
<p>Sheldon: No, you can&#8217;t reprogram people. To James Tiberius Kirk. <em>(Drinks. Leaves. Comes back and spits it out again.)</em></p>
<p><em>Scene: The cafeteria.</em></p>
<p>Sheldon: Hello, all.</p>
<p>Raj: Hello.</p>
<p>Leonard: Hey.</p>
<p>Sheldon: Howard, you&#8217;re feeling better about me today, aren&#8217;t you?</p>
<p>Howard: Not really.</p>
<p>Sheldon: Yes, you are. I&#8217;m using neurolinguistic programming to modify your thought patterns.</p>
<p>Howard: Oh. Go away, Sheldon.</p>
<p>Sheldon: There&#8217;s a nine ninety five e-book down the drain.</p>
<p>Raj: What&#8217;s in the bag?</p>
<p>Sheldon: It&#8217;s for Howard.</p>
<p>Howard: Oh. Sheldon, you can&#8217;t fix this with gifts.</p>
<p>Sheldon: Nevertheless, I&#8217;ve hurt you, and whether you forgive me or not, I want you to have this.</p>
<p>Howard: You&#8217;re giving me a couch cushion?</p>
<p>Sheldon: No. The cushion is merely symbolic. I&#8217;m giving you my spot on the couch.</p>
<p>Howard: But you love that spot.</p>
<p>Sheldon: No. I love my mother. My feelings for my spot are much greater. It is the singular location in space around which revolves my entire universe. And now it&#8217;s yours.</p>
<p>Raj: Oh, my God, dude. Now you have to forgive him.</p>
<p>Howard: All right. Apology accepted.</p>
<p>Sheldon: High five. Not too hard. Thank you.</p>
<p>Raj: I haven&#8217;t cried like this since Toy Story 3.</p>

<p>Howard: I got to tell you, Sheldon, I understand why you chose this spot. I mean, the temperature is good, but there&#8217;s no draft. I can see the television, but I can still talk to&#8230;</p>
<p>Sheldon: I changed my mind. Get out of my spot.</p>
<p>Penny: How long?</p>
<p>Leonard: 94 seconds.</p>

				if ( jetpackLikesWidgetQueue.length > 0 ) {
					// We may have a widget that needs creating now
					var found = false;
					while( jetpackLikesWidgetQueue.length > 0 ) {
						// Grab the first member of the queue that isn't already loading.
						wrapperID = jetpackLikesWidgetQueue.splice( 0, 1 )[0];
						if ( jQuery( '#' + wrapperID ).hasClass( 'jetpack-likes-widget-unloaded' ) ) {
							found = true;
							break;
						}
					}
					if ( ! found ) {
						setTimeout( JetpackLikesWidgetQueueHandler, 500 );
						return;
					}
				} else if ( jQuery( 'div.jetpack-likes-widget-unloaded' ).length > 0 ) {
					// Grab any unloaded widgets for a batch request
					JetpackLikesBatchHandler();

					// Get the next unloaded widget
					wrapperID = jQuery( 'div.jetpack-likes-widget-unloaded' ).first()[0].id;
					if ( ! wrapperID ) {
						// Everything is currently loaded
						setTimeout( JetpackLikesWidgetQueueHandler, 500 );
						return;
					}
				}

				if ( 'undefined' === typeof wrapperID ) {
					setTimeout( JetpackLikesWidgetQueueHandler, 500 );
					return;
				}

				var $wrapper = jQuery( '#' + wrapperID );
				$wrapper.find( 'iframe' ).remove();

				if ( $wrapper.hasClass( 'slim-likes-widget' ) ) {
